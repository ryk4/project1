    Recieps html-template Pohlobkin

    Short intro

    Once I thought that I haven't seen a beautiful recipes theme, so I decided to make it on my own.

    As you can see the most attention is paid to design card. To make it realy responsive and easily to use i use bootstrap #4. With this framework you can customize this template as you want.

    About template

    I am working with this template. It seems to me like a beautiful base to make your own cooking blog.

    So i made some unusual things to make it RTS(Ready To Start):

    • For outer links I check data href and if it start with http or https I mark it as outer with FA-icon.
    • To better use and good optimisation I use good version of .htaccess. It contains rules for Apache server to make your site more powerful.
    • I made all images small as i could for better rate in pages speed test from google.
    • Also I recommend you to use this rules (rel='nofollow' target='_blank') for outer links. It's really good for your SEO.

    Gulp

    I made it template using these soft and languages:

    • NPM
    • Bower
    • Gulp
    • SASS
    • Pug

    To customise this template you just need to get root folder. If you have npm installed, throw this, else - install it. Then open Git Bash or other console run npm install and wait. After all components installed run gulp watch to watch all changes you make. If you want to customize gulp - just open gulpfile.js in root folder.

    Good luck to customize it)

    Bootstrap template

    As I said I used bootstrap #4. And there are a lot of good improves I used in this template:

    • Flexible grid
    • Text utilites
    • Responsive helpers
    • A lot of other utilities

    I put all sass (and scss) rules in one compressed file for better optimisation and couse i used its veriables into my css. To get control you can use gulp rules and make changes in all sass (and scss) files to get your own template with another design.
