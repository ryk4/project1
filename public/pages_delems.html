Nav pills

You can use bootstrap default navigation pills code:

<ul class="nav nav-pills">
   <li class="nav-item">
      <a class="nav-link active" data-toggle="tab" href="#ex1" role="tab">Example 1</a>
   </li>
   <li class="nav-item">
      <a class="nav-link" data-toggle="tab" href="#ex2" role="tab">Example 2</a>
   </li>
</ul>
<div class="tab-content">
   <div class="tab-pane active fade show" id="ex1" role="tabpanel">...</div>
   <div class="tab-pane fade" id="ex2" role="tabpanel">...</div>
</div>
