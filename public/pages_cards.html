Cards

This template was written mostly for these cards. Good cards - are the face of your listing site\/web application.

Main feature of this cards - is flexible design. You don't need to think how place all user-input text. Just make pre-text short in one string. Using cards like in examples - with bootstrap classes .row and .col-* you can get this effect.

There are different types of cards design
  • Default card
  • Card with background image
  • Card that contains only text
  • Card with accent color in background

    • Of course you can customize this template using Gulp in dev folder.

    Cards are contains:

    • Body of card - .b-card_inner
      • Image section .b-card_image
      • Description section section .b-card_description
    • Info sections
      • Meta info section - can be placed in image section and in description)
      • Title and pre-text section - also can be placed in image and description section
      • Autor - usually placed in description section
    • Action buttons
      • Comment
      • Bookmark
